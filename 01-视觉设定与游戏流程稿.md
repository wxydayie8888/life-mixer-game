# 《人生调音台》视觉设定与游戏流程稿

版本：MVP v0.1

定位：一个给所有普通人玩的 5 分钟自我显影网页小游戏。它像游戏，像社交测试，也像一张能转发的个人说明书。它不做心理诊断，不给人贴死标签，而是帮玩家把自己的主旋律、底噪、边界和未来方向调清楚。

核心体验句：

> 这不是测试你是谁，是把你的主旋律调出来。

## 1. 设计关键词

### 必须有

- 酷：像一台可以玩的电子音乐装置，而不是问卷表单。
- 温暖：让玩家愿意承认自己的喜欢、讨厌和脆弱。
- 幽默：结果文案要有轻微吐槽感，让人愿意转发。
- 年轻：有社交测试的爽感，但不廉价。
- 准：问题必须具体，结果必须能被玩家带回真实生活验证。

### 必须避免

- 老式心理测试感：长题干、标准量表、严肃打分、人格判词。
- 星座感：玄学式总结、说什么都像对、没有证据。
- 过度 MBTI 化：不要把玩家硬塞进 16 型人格。
- 心理咨询感：不要像诊断报告，也不要替玩家解释创伤。
- 幼稚卡通感：吉祥物可爱，但界面不能低龄。
- 纯赛博科技感：不要冷冰冰，必须有人味。

## 2. 世界观

玩家进入一间“人生录音棚”。每个人都有一张自己的调音台，上面有还没被听见的音轨：

- 心动音轨：我靠近什么。
- 底噪音轨：什么会消耗我。
- 边界音轨：别人怎样对我，我会更舒服。
- 未来音轨：我愿意成为哪种人。
- 说明书音轨：如何更好地使用我。

游戏过程就是一次快速调音：玩家通过选择生活场景，让不同音轨亮起来。最后系统生成《我的个人使用说明 v0.1》。

## 3. 吉祥物设定

### 首选角色：BPM 小狗

BPM 是音乐里的节拍，也暗示玩家自己的生活节奏。小狗比小猫更适合这个产品：它有陪伴感、朋友感和社交感，更容易承担幽默吐槽。

角色统一母版：

- 三视图文件：[assets/bpm-dog-character-sheet.png](assets/bpm-dog-character-sheet.png)
- 游戏头像文件：[assets/bpm-dog-avatar.png](assets/bpm-dog-avatar.png)
- 首屏主视觉文件：[assets/life-mixer-hero.png](assets/life-mixer-hero.png)

后续所有页面、分享卡、表情包、加载动画都应以三视图为准：金棕色立耳小狗、奶油色口鼻和胸口、黑色监听耳机、深海军蓝连帽外套、暖橙色滚边、BPM 金色吊牌。不要再使用黑棕色垂耳版本，避免角色分裂。

### 外形

- 一只短腿小狗，身体偏圆，但不是幼儿风。
- 戴着一副偏大的监听耳机。
- 耳机线可以像尾巴，也可以接在调音台上。
- 胸前挂一个小小的节拍器吊牌，上面写 BPM。
- 眼睛灵，表情丰富，嘴角有一点“我听懂了”的坏笑。
- 身上可有一块亮色小围巾或小马甲，用来和界面高光色呼应。

### 性格

- 不当老师，不当心理咨询师。
- 像一个懂音乐、懂你一点点、但不会装作完全懂你的朋友。
- 负责提示、吐槽、缓冲和鼓励。

### 语气

可爱但不装可爱，温柔但不鸡汤。

示例：

- “这条音轨亮了。”
- “你不是难搞，这个环境确实有点刺耳。”
- “我听见了：你需要的不是更多鸡血，是更清楚的意义。”
- “这里有底噪，我们先别假装它不存在。”
- “你的能量不是少，是被错误的插座偷走了。”

## 4. 视觉系统

### 整体风格

电子人生调音台 + 漫画小剧场。

主界面像一台电子音乐设备，问题卡像一格格生活漫画，结果页像一张可以发朋友圈的个人专辑封面。

### 色彩

底色：

- 录音棚黑：#111217
- 深夜蓝黑：#171B2A
- 控台灰：#252933
- 柔雾灰：#E8E5DC

高光色：

- 电光青：#20E6D2，用于心动、探索、解释。
- 节拍绿：#B7F34A，用于完成、确认、亮起。
- 暖橙：#FFB84D，用于温暖提示、恢复方式。
- 桃粉：#FF6FAE，用于表达、关系、分享。
- 警报红：#FF5C5C，用于底噪、消耗、过载。

颜色原则：

- 不做单一紫蓝科技风。
- 每个维度有固定颜色，方便结果页形成记忆。
- 深色背景必须配足留白和柔和阴影，避免压抑。

### 字体气质

中文主标题：建议使用有一点标题感、但不古板的黑体。可参考思源黑体 Heavy、阿里巴巴普惠体 Heavy、HarmonyOS Sans SC Bold。

正文：清晰、圆润、易读。可用思源黑体、阿里巴巴普惠体、霞鹜文楷仅用于少量手写感注释。

英文与数字：可选 Space Grotesk、IBM Plex Mono、Inter。数字和百分比可以用等宽字体，增强设备感。

### UI 形态

- 卡片圆角：8px 左右，不要大胶囊。
- 按钮：优先图标 + 短文案，比如靠近、无感、远离。
- 调音台推子：竖向或横向都可，MVP 建议横向，移动端更好操作。
- 波形：轻动画，不要过度炫技。
- 漫画卡：黑色细线 + 局部高光色，像快速生活分镜。

### 首屏结构

中间：

### 首页状态

- 心动
- 底噪
- 边界
- 未来
- 说明书

玩家点击开始后，BPM 小狗说：

“好，别端着。我们听听你真正会被什么点亮。”

## 6. 游戏流程

MVP 总时长控制在 4 分 30 秒到 6 分钟。

### 第 0 步：开机校准

时长：15 秒。

问题：

你今天为什么想调一调自己？

选项：

- 我有点迷茫
- 我想更了解自己
- 我最近有点耗电
- 我想生成一份能给别人看的说明书

作用：

不参与核心人格计分，只影响结果页的开场语气。

### 第 1 步：心动采样

时长：90 秒。

玩法：

连续出现 18 张生活漫画卡。玩家选择：

- 靠近
- 无感
- 远离

每张卡背后关联 1 到 2 个主旋律维度。靠近加分，远离少量反向记录，不用于惩罚。

动效：

玩家点“靠近”时，对应音轨亮起一格；点“远离”时，卡片轻微失真，底噪屏闪一下。

BPM 小狗提示：

- “这不是兴趣，这是你的电量入口。”
- “有些事情没错，只是不通你的电。”

### 第 2 步：底噪消除

时长：75 秒。

玩法：

出现 12 组二选一。题目是“如果必须忍一个，你更不能忍哪个？”

每组选项分别对应不同压力触发器。被选中的一方记为更强底噪。

动效：

两张卡像两个声音源，玩家选择后，另一个被静音，被选中的底噪进入“待处理轨道”。

BPM 小狗提示：

- “讨厌不是坏脾气，有时候是边界在报警。”
- “你不用喜欢所有环境，真的。”

### 第 3 步：未来试音

时长：90 秒。

玩法：

玩家看到 8 个未来身份卡，每张卡都包含“你会得到什么”和“你要付出什么代价”。玩家选择最愿意走进去的 3 个。

动效：

每选择一个身份，舞台灯亮起，人物剪影进入录音棚。

BPM 小狗提示：

- “想成为谁，关键不是选光环，是选你愿意承担的代价。”
- “这一步不看你羡慕什么，看你愿意长期靠近什么。”

### 第 4 步：结果调音

时长：45 秒。

玩法：

玩家看到调音台自动混音：

- 三条最亮主旋律
- 三个主要底噪
- 一个未来主轴
- 一句“我正在成为的人”

玩家可以点：

- 很像我
- 有一点不像

如果点“有一点不像”，系统提供三个可替换文案，不重新测试。

### 第 5 步：生成说明书

时长：30 秒。

生成内容：

- 《我的个人使用说明 v0.1》
- 三张可分享卡
- 一段可复制的朋友圈文案

## 7. 结果页设计

### 结果页顶部

标题：

我的个人使用说明 v0.1

副标题：

由 BPM 小狗在你的调音台旁边认真听完后生成。

### 核心可视化

一张“人生混音面板”：

- 主旋律：前三个维度，用音轨条展示。
- 底噪源：前三个压力触发器，用噪点标签展示。
- 未来方向：一个身份卡片。
- 当前版本：一句人格叙事。

### 结果文案模块

1. 我会被这些事情点亮
2. 我会被这些事情消耗
3. 我低电量时会变成这样
4. 如何更好地使用我
5. 我正在成为的人
6. 30 天小实验

### 修正机制

每个模块底部有一个小按钮：

“这句不太像我”

点击后出现 3 个替代表达。MVP 不需要复杂编辑器，只需要允许替换。

## 8. 分享卡设计

### 分享卡 1：专辑封面版

用途：朋友圈传播主卡。

标题：

我的人生主旋律

视觉：

像一张电子音乐专辑封面。深色背景，中间是三条发光音轨，BPM 小狗在角落戴耳机。

内容：

- 主旋律：创造 / 解释 / 探索
- 底噪：被打断 / 无标准 / 空转
- 当前版本：把混乱变清楚的人

底部：

人生调音台 | 5 分钟生成你的个人使用说明

### 分享卡 2：个人使用说明版

用途：适合发给朋友、伴侣、同事。

标题：

如何更好地使用我

视觉：

像一张设备说明书，但更年轻。左侧是 BPM 小狗拿着小扳手，右侧是说明条目。

内容：

- 请给我：清晰目标、真实反馈、安静时间
- 请避免：临时打断、含糊评价、无意义重复
- 我低电量时：会沉默，不是不在乎

底部：

当前版本：v0.1，欢迎温柔更新。

### 分享卡 3：BPM 吐槽版

用途：最年轻、最容易被转发。

标题：

BPM 小狗听完后说：

视觉：

漫画大表情。小狗戴耳机，一脸“我懂但我不乱说”的表情。

内容示例：

- “你不是懒，你是对没有意义的事情电量过敏。”
- “你的主旋律很明显：想做点真正属于自己的东西。”
- “你不是不合群，你只是不能长时间待在低质量噪音里。”

底部：

人生调音台

## 9. MBTI 连接方式

原则：借传播性，不借决定论。

不要输出：

“你就是 INFP。”

可以输出：

“如果你熟悉 MBTI，你可能会在自己身上看到一点 N 型的想象力，或 J 型的结构需求。但《人生调音台》更关心的是：你在什么场景里会发亮，以及什么环境会让你失真。”

MVP 中可以做一个轻连接模块：

标题：

和 MBTI 的一点点对照

文案：

“这不是 MBTI 类型判断，只是给熟悉 MBTI 的你一个参考坐标。”

对照维度：

- 探索/解释/创造较高：可能更接近 N 倾向。
- 连接/表达较高：可能更接近 E 或 F 的外显表现。
- 秩序/组织较高：可能更接近 J 倾向。
- 独处打磨/深度研究较高：可能更接近 I 的能量恢复方式。
- 决策/结构/问题解决较高：可能更接近 T 倾向。
- 照护/关系/意义较高：可能更接近 F 倾向。

注意：结果页必须写“可能”，不能写“你是”。

## 11. MVP 开发优先级

### P0 必须做

- 三段选择流程
- 主旋律计分
- 底噪计分
- 未来身份选择
- 结果页
- 至少一张分享卡

### P1 尽量做

- BPM 小狗表情变化
- 三张分享卡
- 结果文案替换
- 简单波形动效

### P2 以后再做

- 登录保存
- 好友对比
- 完整 MBTI 对照
- 题库个性化推荐
- AI 生成更长说明书
