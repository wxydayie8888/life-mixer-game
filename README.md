# 人生调音台

一个 5 分钟自我显影网页小游戏。

它首先是一个适合转发的社交小游戏，其次才是个人成长工具。玩家通过心动采样、底噪消除和未来试音，生成一份《我的个人使用说明 v0.1》。

## 当前 MVP

- 电子人生调音台 + 漫画小剧场视觉风格
- BPM 小狗吉祥物和三视图角色设定
- 18 张心动采样题，配 6 张统一风格漫画场景
- 12 组底噪二选一
- 8 个未来身份
- 自动生成个人使用说明
- 结果分享卡、下载专辑分享卡、复制朋友圈文案

## 本地预览

```bash
python3 -m http.server 8766
```

然后打开：

```text
http://localhost:8766
```

## 产品原则

- 不做心理诊断
- 不输出固定人格标签
- 不把 MBTI 当结论
- 用具体生活场景和边界选择提高“准”的感觉
- 让玩家愿意继续点、愿意截图、愿意发给朋友
